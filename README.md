## AndroidSDKPull
AndroidSDKPull By DotNet Core, for mirrors.zzu.edu.cn

##功能
*   可从dl.google.com拉取Android SDK数据到本地
*   采用.net core开发，可支持跨平台安装部署
*   支持自定义文件配置

* * *

##安装使用
*   安装dotnet运行环境 https://www.microsoft.com/net/core#linuxcentos
*   配置程序的config.json
*   执行```dotnet AndroidSDKPull.dll``` 运行即可
*   加入系统自动任务crond定时运行
